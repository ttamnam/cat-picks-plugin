# Cat Picks Plugin

## Overview
This plugin was created for the WJCT WordPress skills test.

The plugin registers a custom post type called "Cat Picks" and adds a custom meta field called "Featured By."

## Features
- Custom post type registration
- Custom meta box in WordPress admin
- Ability to save Featured By values
- Frontend display of Featured By data
- Uses WordPress sanitization and escaping functions

## Installation
1. Copy the plugin folder into:
   wp-content/plugins/
2. Activate the plugin from:
   Plugins → Installed Plugins

## Testing
1. Activate the plugin
2. Create a new Cat Pick post
3. Enter a value into the Featured By field
4. Publish the post
5. Verify the Featured By value appears on the frontend
